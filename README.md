# React Burger Builder

A React learning project that lets users build a custom burger by adding and removing ingredients dynamically.

## Features

- Add and remove burger ingredients
- Dynamic UI updates based on selected ingredients
- Ingredient price calculation
- Conditional rendering for order state
- Modular component structure

## Tech Stack

- React
- Redux
- React Router
- Vite
- JavaScript
- CSS

## Project Structure

- `src/components`
- `src/containers`
- `src/assets`
- `src/App.js`
- `src/index.jsx`

## Run Locally

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build production bundle:

```bash
npm run build
```

The dev app runs at `http://localhost:3000`.
